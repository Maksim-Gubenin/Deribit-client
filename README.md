# Deribit Client Project

## Описание:
Клиент для криптобиржи Deribit (https://docs.deribit.com/). Клиент каждую минуту забирает с биржи текущую цену btc_usd и eth_usd (index price валюты) после чего сохраняет в базу данных тикер валюты, текущую цену и время в UNIX timestamp. 

Имеет внешнее API для обработки сохраненных данных на FastAP

## Структура API:
- Получение всех сохраненных данных по указанной валюте
- Получение последней цены валюты
- Получение цены валюты с фильтром по дате
